# HabitFlow Tracker

Ein moderner, PWA-fähiger Habit Tracker mit Fokus-Timer und Streak-System.

## Features
- **Daily Tasks**: Verwalte deine täglichen Aufgaben.
- **Handyzeit-Logger**: Setze dir ein Limit und logge deine Zeit (manuell).
- **Streak System**: Halte deine Strähne, indem du täglich alles erledigst.
- **Progressive Web App (PWA)**: Installierbar auf Android/iOS.
- **Offline-Fähig**: Funktioniert auch ohne Internet.

## Installation & Start

1.  **Voraussetzungen**: Python installiert.
2.  **Abhängigkeiten installieren**:
    ```bash
    pip install flask
    ```
3.  **App starten**:
    ```bash
    python app.py
    ```
4.  **Im Browser öffnen**:
    Gehe zu `http://localhost:5000`

## Als App installieren (Handy)
1.  Stelle sicher, dass dein Handy und PC im gleichen WLAN sind.
2.  Finde die IP-Adresse deines PCs heraus (z.B. `ipconfig` -> IPv4).
3.  Öffne am Handy den Browser und gehe zu `http://YOUR_PC_IP:5000`.
4.  Tippe auf "Zum Startbildschirm hinzufügen" oder "Installieren".
